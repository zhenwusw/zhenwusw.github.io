 > Elixir is a functional meta-programming aware language built on top of the Erlang VM. It is a dynamic language with flexible syntax with macros support that leverages Erlang's abilities to build concurrent, distributed, fault-tolerant applications with hot code upgrades. 


Just as [Benjamin](http://benjamintanweihao.github.io/blog/2013/06/08/why-my-next-programming-language-is-elixir/)
said: Having a pretty syntax wrapped around the powerful Erlang VM seems to be
an irresistible combination, that's why Elixir might become the next language
you would like to learn :)
